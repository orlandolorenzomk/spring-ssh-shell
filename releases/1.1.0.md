# 1.1.0

## New

- `SshShellServer.isRunning()` — check whether the server is currently started
- `SshShellServer.getPort()` — retrieve the port the server is listening on

## Fixed

- **Security** — host key file is now written with `600` permissions (owner read/write only) on POSIX systems, preventing other users from reading the private key
- **Security** — single-user password comparison is now constant-time (`MessageDigest.isEqual`) to prevent timing side-channel attacks
- **Banner rendering** — newlines in custom banners are normalized to `\r\n` before printing; bare `\n` in SSH raw mode caused a staircase effect in multi-line banners
- **Escape sequences** — arrow keys, function keys, and Delete no longer corrupt the input line buffer; the full CSI escape sequence is consumed and discarded
- **Logging** — a warning is now logged when the host key file is corrupted and regenerated, explaining why clients will see a host key mismatch
- **Error stream** — `setErrorStream` now stores the stream instead of silently discarding it

## Dependency changes

- `slf4j-jdk14` scope changed from `compile` to `runtime` — consumers no longer receive it as a transitive compile dependency, preventing conflicts with their own logging setup
