# 1.0.0 — Initial Release

## Features

- Embeddable SSH shell server built on Apache MINA SSHD
- Fluent builder API (`SshShellServer.builder()`)
- Single-user authentication via `.username()` / `.password()`
- Multi-user / custom authentication via `.authenticator(SshAuthenticator)`
- Custom command handler via `.commandHandler(ShellCommandHandler)`
- Host key persistence across restarts via `.hostKeyFile(Path)`
- Idle timeout via `.idleTimeoutSeconds(int)`
- Custom welcome banner via `.banner(String)`
- Custom executor via `.executorService(ExecutorService)`
- Backspace / DEL support in the shell
